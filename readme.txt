# Radius Digital - Local SEO Landing Page

This is a responsive landing page built for **Radius Digital**, a business that helps UK companies optimise their Google Business Profiles for better visibility and lead generation.

## 🚀 Features
- Fully responsive layout using Tailwind CSS
- SEO-friendly meta tags and title
- Clear CTA sections (Problem → Solution → Offer → Contact)
- Contact form integration via Formspree

## 📁 Folder Structure
```
project-root/
├── index.html
├── README.md
├── sitemap.xml
└── robots.txt
```

## 🛠️ Setup & Deployment
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/radius-digital-seo.git
   ```

2. Replace the Formspree form ID in `index.html` contact form:
   ```html
   <form action="https://formspree.io/f/YOUR-FORM-ID" method="POST">
   ```

3. Deploy to Vercel, Netlify, or GitHub Pages.

---

## 🌐 SEO & Performance
- Meta tags for description, keywords, author
- Preconnect and DNS prefetch for CDN
- Tailwind script is deferred to reduce render-blocking
- Suggest using Tailwind CLI for production build and self-hosting assets for faster load
